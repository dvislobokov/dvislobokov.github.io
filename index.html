sconf

Layered configuration, modeled after ASP.NET Core. JSON, YAML, TOML, env vars, CLI flags and Vault secrets merge per key into one typed struct — including arrays of objects built from environment variables.

sorm

The Go ORM with a real Unit of Work. Track entities, mutate plain structs, call SaveChanges once — diffs, write ordering and optimistic concurrency are handled for you. PostgreSQL, MySQL, SQLite.

err := session.SaveChanges(ctx)

srog

Serilog-style message templates for Go, on top of zerolog. Write the message once, get typed structured fields for free — with zero allocations on the hot path, rotation, ECS and OpenTelemetry output.
